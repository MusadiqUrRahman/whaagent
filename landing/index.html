Architecture

Three-node state graph.

Summarize, Router, Agent. Token-aware compression, LLM-powered intent classification, and tool execution in a single directed graph.

Summarize
Router
Agent
Go Gateway

WhatsApp session management, QR code generation, WebSocket connection with automatic reconnection.

FastAPI Server

Multi-tenant agent pools with LRU eviction, keep-alive health checks, per-tenant isolation.

MCP Protocol

Model Context Protocol with circuit breaker resilience. Auto-discovers tools, graceful degradation.

Agents

12 specialized agents.

Each agent has its own system prompt, tool set, and MCP connections. The assistant agent orchestrates sub-agents via graph-based delegation.

Quickstart

Running in 60 seconds.

Docker
$ git clone https://github.com/MusadiqUrRahman/whaagent.git
$ cd whaagent && cp .env.example .env
# add your API key to .env
$ docker compose up -d
$ curl http://localhost:8000/health
Manual
# Python 3.12, Go 1.21+
$ uv sync
$ cd gateway && go build -o whaagent-gateway
$ whaagent serve
CLI
$ whaagent list
12 registered agents
$ whaagent chat "Hello" -a assistant
Agent responds via configured LLM provider
