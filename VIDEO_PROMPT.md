# whaagent — Video Generation Prompt

## Recommended Model

**Runway Gen-4** (https://runwayml.com) — Best for motion graphics, precise camera control, and professional tech product videos. $15/mo Standard plan. Supports image-to-video, motion brush, and camera path controls.

**Alternative:** Kling 3.0 (https://klingai.com) — Better for longer clips (up to 15s), more cost-effective ($10/mo), strong multi-shot sequences with synced audio.

**Avoid:** Sora 2 — app shut down April 2026, API sunset September 2026.

## Assembly Notes

1. **Generate each shot** individually on Runway Gen-4 (or Kling 3.0)
2. **Color grade** all shots to match — consistent green (#25D366) accent, dark backgrounds
3. **Cut on action** — transition between shots at moments of movement (pulse, type, orbit)
4. **Add audio** in post — subtle electronic/ambient soundtrack, soft keyboard clicks for terminal shot, notification sound for WhatsApp shot
5. **Total duration:** ~30-36 seconds (6 shots × 5-6 seconds each)
6. **Aspect ratio:** 16:9 for YouTube/website, 9:16 for social media (re-generate with adjusted framing)
